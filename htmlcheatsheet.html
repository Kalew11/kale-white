HTML Cheat Sheet

  • Always make sure to close tags!
  • h1 to h5 are used to quickly select the size of font you want before editing it further, also helps with targetting with css
  • Create divs to allow for easy targeting for styling in CSS, as well as targeting values in Javascript!
  • Although it is best to style in CSS, "<b>, <i>, <u>" are useful to include, as it will give you a rough outline, or can be used to fully style pages where formatting isn't used as much, such as resume pages which need to be kept clean
  • use ctrl + / to comment out things
  • always make sure to size images properly using height or width. Putting one will keep the image proportions proper. Putting many may skew the image size
  • use " " for quick formatting if just a few spaces are needed.
  • keep tags consistent across pages so that they are easy to style with css. ie. h1 tags can be used for titles so they both style the same
