JavaScript Cheat Sheet

  • use var and let to declare variables in JavaScript. Unlike C++, we do not need to declare values for them such as int and float
  • JavaScript includes useful events such as onclick(), onhover() to implement functions when these events are called
  • use document.getElementById and document.getElementsByClassName as two common methods to target id's and classes. Note that it is getElement for one and getElements for the other one. Not realizing this can create some confusing moments.
  • use = if you are setting something to be that value, and == or === if you want to ask if it is that value.
  • Place scripts at the bottom of the page to allow the website to load the data first. Exceptions to this can be sites such as youtube, where the scripts loading first play a key role in allowing navigation.
  • JavaScript has a keyword called this, which returns values from the object it belongs to.
  • JavaScript has a .innerHTML property that will return the value inside the div. So if the div was named demo, and there was a paragraph inside, you could use document.getElementById("demo").innerHTML to target that area

Some frequently used methods from the link above

  • concat() Joins two or more arrays, and returns a copy of the joined arrays
  • join() Joins all elements of an array into a string
  • pop() Removes the last element of an array, and returns that element
  • push() Adds new elements to the end of an array, and returns the new length
  • shift() Removes the first element of an array, and returns that element
  • unshift() Adds new elements to the beginning of an array, and returns the new length
